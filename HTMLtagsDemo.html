<html> Tag

The <html> tag is the root of an HTML document. It wraps the entire web page code.

<head> Tag

The <head> tags contains meta-info about the document such as title, character set, links to stylesheets, etc.

<title> Tag

The <title> tags defines the title of the page. You can see it in the browser tab. For this page, it is set as: HTML Tags Demo- Tech.

<body> Tag

The <body> tags contains all the visible content of the web page like headings, text, images, links, lists, etc. Everything you see here is inside the <body> tag.
