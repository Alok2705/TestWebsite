Home page

In HTML, both <ul> (unordered list) and <ol> (ordered list) are used to display lists. The main difference is that:

  • <ul> shows the lists items with bullets
  • <ol> shows the lists items with numbers or letters(1,2,3 or A,B,C)

Unordered list example- features of tech

  • weekend live online classes
  • hands-on coding practice
  • industry-level project guidance
  • job-oriented course design

Ordered list example- steps to enroll at Tech

  1. Visit our official website
  2. choose your course(e.g. Python, Web Development)
  3. fill the registration form
  4. attend the orientation session
  5. start learning with live classes

Summary

  • <ul> = unordered list(bullets)
  • <ol> = ordered list(numbers or letters)
  • <li> = defines each item in both types of lists

Tech uses both types of lists on its website to present structured and easy to read info to students.
